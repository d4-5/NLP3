
# Cleaning (очистка)

Що очищаємо

- множинні пробіли → один пробіл
- пробіли перед пунктуацією  
- табуляції → пробіл
- не-розривний пробіл (`\xa0`)
- зайві переноси рядків
- пробіли на початку та в кінці тексту

Що не змінюємо

- регістр (lowercasing не застосовується)
- числа
- імена власні
- граматичні форми
- порядок слів
- пунктуацію як семантичний елемент

# Normalization (нормалізація символів)

Нормалізуємо

Unicode-варіації символів:
- тире `–`, `—` → `-`
- лапки `« » “ ”` → `"`
- різні апострофи `’`, `‘` → `'`

Не нормалізуємо

- літери (кирилиця/латиниця не перетворюється)
- числа
- морфологію
- орфографію

# Masking (PII)

- URL → `<URL>`
- Email → `<EMAIL>`
- Phone → `<PHONE>`


# Sentence Splitting

Sentence splitting реалізований rule-based способом.

Розділювачі речень

- `.`
- `!`
- `?`

Не розбиваємо після скорочень

- `м.`
- `р.`
- `т.д.`
- `ім.`

Не розбиваємо

- десяткові числа (3.14)
- версії (1.2.3)
- конструкції, де крапка не означає кінець речення
